Inspector!

O Inspector é um aplicativo para dispositivos Android desenvolvido utilizando o Android Studio tendo como intuito facilitar o processo de inspeções radiológicas realizadas por Físicos Médicos. Para autenticação, banco de dados e armazenamento de imagens foi utilizado o Firebase.
